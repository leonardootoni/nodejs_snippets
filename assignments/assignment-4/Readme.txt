1 - Create a npm project and install Express.js and Nodemon

2 - Install a preferrable template engine like Handlebars, Pug or EJS

3 - Add two routes:
    - "/" => Holds a <form> that allows users to input their name
    - "/users" => Outputs an <ul> with the user names (or some error text)

4 - Implement a solution to store the names and render the names in an unordered list.
    for this task, considered to use a template engine as well.
